## Overview
The Network Traffic Load Generator is a Python-based, multi-threaded tool designed to simulate TCP, UDP, and HTTP traffic for educational, testing, and research purposes.
It allows users to generate configurable traffic to analyze server performance, network behavior, and rate-limiting mechanisms in controlled environments.

Key Points:
- Supports TCP, UDP, and HTTP request generation
- Adjustable packets/requests per second
- Multi-threaded for high-volume simulations
- Intended strictly for ethical testing on systems you own or have explicit permission to test
- Helps network engineers, researchers, and students study server/network behavior under load

## Features

- Configurable load profiles for TCP, UDP, and HTTP.
- Adjustable connection/request rates and concurrency.
- Dry-run / simulation mode (no traffic sent) for validating configurations.
- Simple reporting/logging to help analyze results.
- Safety checks and target whitelisting to reduce risk of accidental misuse.
- Extensible design so defensive testing workflows and mitigations can be integrated.

## Quick start
1. Clone or download this repo to your machine:
   ```bash
   https://github.com/sahikk67/StormX-NetworkTool
   cd StormX-NetworkTool
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the tool:
```bash
python "StormX-NetworkTool.py" 
python StormX-NetworkTool.py -h / --help
```
Ensure you have Python installed on your system.
Usage
To run the DDoS attack tool, use the following command format:
```bash
python StormX-NetworkTool.py <target_ip> <target_port> <target_url> <attack_type> <packets_per_second> <requests_per_second> <num_threads>
```
Examples : 
```bash
TCP DDoS Attack:
python StormX-NetworkTool.py 192.168.1.1 80 http://example.com tcp 1000 500 500
------------------------------------------------------------------------------------
UDP DDoS Attack:
python StormX-NetworkTool.py 192.168.1.1 80 http://example.com udp 1000 500 500
------------------------------------------------------------------------------------
HTTP DDoS Attack:
python StormX-NetworkTool.py 192.168.1.1 80 http://example.com http 1000 500 500
```
